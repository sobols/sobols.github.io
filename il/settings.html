Настройка

Вследствие отсутствия GUI-настройщика все установки производятся путем правки конфигурационного файла.

Чтобы настроить параметры, нужно удедиться, что InductionLines закрыта, затем открыть файл settings.ini в текстовом редакторе (например, Блокноте). Файл имеет стандартную структуру.

Изменяя соотношения между Length и Radius, можно сделать катушку "толще" или "вытянутее". Turns определяет количество витков и может быть нецелым (при 0.000001 катушка практически превращается в прямой проводник и т.п.). Segments и StepLength управляют точностью и качеством картинки.

Любая линия может быть описана ограниченным числом точек. Это число задается параметром MaxVertex. Поэтому иногда за MaxVertex шагов программе не удается получить замкнутую линию, и получается линия, не дорисованная до конца. Такие линии можно не выводить на экран, если DrawCorrupted установить в 0.

По обьективным причинам линии обычно не получаются замкнутыми. Иногда ситуацию можно исправить установкой параметра AutoConnect в 1.

Максимальная длина значения каждого параметра - 15 символов. НЕ БОЛЬШЕ!

Ввод значений не проверяется в программе на корректность. Эксперименты с очень большими/малыми числами могут закончиться полным зависанием.

Пример файла settings.ini

[Solenoid]
Length=5.0
; длина катушки (в условных единицах) - действительное число

Radius=1.0
; радиус катушки - действительное число

Turns=20
; количество витков - действительное число

Segments=2000
; на сколько элементарных частей при расчете индукции будет разбита катушка - натуральное число

[Lines]
LineNumber=6
; количество линий (быстрее работает с четными числами, т.к. при нечетном числе одна линия будет выходить точно из центра катушки) - целое неотрицательное число

StepLength=0.01
; длина малого отрезка линии (чем меньше, тем точнее) - действительное число

MaxVertex=10000
; макс. количество отрезков, из которых может состоять одна линия (натуральное число), или сколько вершин (vertex) будет выделено при построении одной линии в вершинном буфере. Не следует перебарщивать с этим числом.

DrawCorrupted=1
; если 0, то не дорисованные до "возвращения" в катушку линии не выводятся на экран. Значения - только 0 и 1.

AutoConnect=1
; в несложных случаях позволяет замыкать линии автоматически (работает только если конец линии находится в катушке). Значения - только 0 и 1.

[View]
AngleStep=0.017453292
; шаг изменения углов (в радианах!). Уменьшение поможет, если при нажатии стрелок углы обзора меняются слишком быстро. Действительное число.

DistanceStep=0.01
; шаг изменения расстояния (в условных единицах). Уменьшение поможет, если при нажатии PgUp/PgDown расстояние меняется слишком быстро. Действительное число.
