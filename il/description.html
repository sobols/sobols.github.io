Описание

    Основные фичи (features):
  • Использование API Microsoft DirectX позволяет получить достаточно высокую производительность в 3D. Используются аппаратные возможности видеокарты.
  • Возможность посмотреть с разных сторон, с разных расстояний и под разными углами на катушку и линии индукции с эффектом перспективы! Изменение углов и расстояний происходит в реальном времени с помощью клавиш клавиатуры.
  • Два источника света: точечный (point) и направленный (directional). Управление яркостью света в реальном времени. Точечный источник расположен в центре катушки и позволяет адекватно оценивать расстояния до линий (яркость линий изменяется обратно пропорционально квадрату расстояния до источника). Направленный источник светит из точки расположения наблюдателя и позволяет четко увидеть объемность катушки.
  • Оконный режим с возможностью произвольного изменения размеров окна.
  • Полноэкранный режим с выбором разрешения и частоты монитора.
  • Вывод (при желании пользователя) на экран FPS, статистики, справки, отладочной информации с использованием эффекта прозрачности текста.
  • Режим автовращения - катушка автоматически крутится вокруг своей оси. Можно использовать в качестве скринсейвера.
  • Вывод статистики построения линий при загрузке.
  • Широкие возможности настройки (с помощью ini-файла).

    Известные недостатки и баги:
  • При построениии используются численные методы расчета. Как известно, операции с действительными числами выполняются компьютером с ограниченной точностью, при выполнении же миллионов операций подряд накапливается заметная ошибка. Поэтому линии индукции обычно получаются незамкнутыми (начало и конец не совпадают).
  • Скорость построения (при определенных настройках) оставляет желать лучшего, хотя по сравнению с предыдущей версией программа работает в два раза быстрее.
  • Отсутствует (пока) графический конфигуратор (все настройки - через ini-файл).
  • Возможны глюки и сбои.

ВНИМАНИЕ!

Для работы требуется DirectX 9. Оттестировано под WinXP Pro SP2, DX9.0c, видеокарты VIA Unichrome Pro (интегрированная), GeForce 6600GT, ATI Radeon 9000. Возможен запуск под GNU/Linux с помощью wine.

Перед запуском программы сохраните все важные данные!

Загрузка программы может длиться довольно долго! Не рекомендуется завершать процесс через "Диспетчер задач".
